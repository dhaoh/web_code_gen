【提交内容】
1. 汇报PPT
用于 Week 13 课堂汇报。PPT 不超过 10 页。
2. 研究报告
报告是本课程重点考核内容，课程无考试。报告不少于 50 页。建议使用 LaTeX 撰写，可参考 ACM / IEEE Transactions 模板（可参考https://pt.overleaf.com/latex/templates/association-for-computing-machinery-acm-large-1-column-format-template/fsyrjmfzcwyy）。

汇报时可以用中文讲，报告和ppt需要用英文撰写。

PPT 和研究报告中均需包含以下内容：
1）选题情况
2）如何使用智能模型驱动方法解决选题方案
3）实现的技术细节

亮点加分项：提出新方法，或对已有方法进行改进

3. 一套可复现、可测试的代码
1）需要提交完整 Git 仓库代码。
2）如果调用 API，需要包含完整可运行代码。
3）如果使用Code Assistant，需要说明 Code Assistant 的使用方式，包括提交 plugin 或 skills 级使用说明。
4）需要包含测试用例，并说明如何运行测试用例。
5）代码应能够通过 CI/CD。
6）后续 GitHub Lab 仓库开放后，大家需按要求上传。
